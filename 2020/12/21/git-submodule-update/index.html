更新git项目中submodule的命令

2020年12月21日
| git
| 阅读

之前一直没有注意submodule如何更新的问题,一直以为我clone后submodule就是新的,后来才发现即便我重新clone,submodule也是按照当时提交时候的版本.找了几个小时才找到简单的命令进行更新submodule.

git submodule update --recursive --remote --merge --force
